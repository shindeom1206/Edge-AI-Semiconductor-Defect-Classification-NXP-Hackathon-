# Edge-AI-Semiconductor-Defect-Classification-NXP-Hackathon-
Edge-AI system for semiconductor defect classification using MobileNetV2 transfer learning. Classifies 8 wafer defect types with 92.87% F1-score. Optimized for low-power edge deployment and exported in ONNX format for NXP eIQ. Includes dataset, training pipeline, evaluation, and Colab-ready notebook.
